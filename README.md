# Explore Early Retirement

An interactive informational tool to help users explore what early retirement could look like by visualizing how life changes based on key variables.

## Features

- **Three Dimensions**: Explore combinations of location, family context, and lifestyle
- **Interactive Selection**: Collapsible sections for easy exploration
- **Dynamic Outcomes**: See how different choices affect your envisioned retirement
- **Minimal Design**: Clean, thoughtful interface focused on clarity

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
